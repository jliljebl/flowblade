Установка с использованием .deb

ШАГ 1. Скачивание и установка .deb

Сначала скачайте пакет .deb с последней версией Flowblade отсюда.

Для установки дважды щёлкните по пакету .deb.

В некоторых системах двойной щелчок может не сработать (на Ubuntu иногда ставится старая версия из репозитория), в этом случае вам придётся установить пакет .deb с помощью терминала:

  • Откройте терминал в папке, в которой был сохранён загруженный пакет .deb. Задайте команду:

    sudo dpkg -i ./flowblade-1.14.0-1_all.deb

ШАГ 2 (ДОПОЛНИТЕЛЬНЫЙ). Дополнительные команды в терминале

Возможно, вам понадобится задать дополнительные команды в терминале:

  • Для установки всех зависимостей:
    • sudo apt-get install -f

Релиз был протестирован на: Ubuntu 16.10, Ubuntu 16.04, Linux Mint 18 и Debian 8. Также должен работать на всех последующих дистрибутивах Debian.

Установка из репозитория вашего дистрибутива

Самый простой способ установки Flowblade — это использовать версию из репозитория вашего дистрибутива. Недостатком является то, что доступная версия может быть устаревшей. Если в репозитории вашего дистрибутива используется старая версия Flowblade или отсутствует вовсе, уведомите об этом его разработчиков.

Установка из исходного кода

Flowblade в настоящее время на 100% является скриптовым приложением, и все зависимости должны быть доступны в популярных дистрибутивах, поэтому в большинстве случаев Flowblade можно запустить без компиляции.

Получить дополнительную информацию можно здесь.

Установка разрабатываемой версии

Flowblade в настоящее время на 100% является скриптовым приложением, и все зависимости должны быть доступны в популярных дистрибутивах, поэтому в большинстве случаев Flowblade можно запустить без компиляции.

Однако разрабатываемая версия может быть нестабильной или иметь новые зависимости.

Для установки разрабатываемой версии, вам нужно установить git и вытащить её из репозитория проекта Github.

Получить дополнительную информацию можно здесь.

Windows и OS X не поддерживаются. Flowblade сосредоточен на предоставлении наилучшей поддержки в Linux и в других свободных платформах.
